 Sistema de Gestión Académica

Imagina que te han encargado la tarea de desarrollar un sistema de gestión académica para un instituto. Deberás crear un conjunto de clases en Java que permitan gestionar estudiantes y cursos, y calcular el promedio de calificaciones de un curso en particular.

    Clase Estudiante:
        Atributos: nombre, número de identificación, array de calificaciones.
        Métodos: constructor, getters.

    Clase Curso:
        Atributos: nombre, array de estudiantes.
        Métodos: constructor, método para agregar estudiantes, método para calcular el promedio de calificaciones.

    Clase MainProgram:
        Método principal (main) para probar el sistema.
        Crear instancias de estudiantes, cursos, y probar las funcionalidades del sistema.

Asegúrate de implementar la lógica necesaria para calcular el promedio de calificaciones de todos los estudiantes en un curso específico.
